# Добро Пожаловать!

Flw.Onl — сервер, где главное это люди, а не гринды!

{% hint style="info" %}
Добро пожаловать на WIKI проекта **FLW.ONL -** тут ты найдешь много полезной и бесполезной информации :)
{% endhint %}

> **FLW.ONL** — ваш приватный, ванильный и полностью бесплатный проект!
>
> Мы сделали сервер с упором на **классический дух Minecraft**, сохраняя его первоначальную атмосферу и механики. Наша философия — минимальное, бережное вмешательство: мы добавляем только те улучшения, которые **не нарушают ванильный игровой процесс**, а лишь делают его чуть удобнее.
>
> Здесь нет платных преимуществ, донат служит только для поддержки сервера, а вайтлист помогает сохранить комфортное сообщество.
>
> Присоединяйтесь, если ищете настоящий ванильный опыт в кругу единомышленников.\
> Рады видеть каждого, кто ценит дух оригинального Minecraft!&#x20;
