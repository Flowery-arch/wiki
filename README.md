# Добро Пожаловать!

**FLW.ONL** — необычный ванильный сервер Minecraft с городами, необычными механиками, ламповой атмосферой и дружной компанией.

{% hint style="info" %}
Здесь собрана актуальная wiki по подтвержденным механикам сервера, командам, правилам и полезным ссылкам.
{% endhint %}

> **IP:** `mc.flowerymc.space`

## Что есть на сервере

Подтвержденные крупные механики проекта:

* города, друзья и репутация;
* локальный и глобальный чат с кастомными командами;
* кастомные крафты и мелкие ванильные улучшения;
* динамические границы мира;
* сезонное открытие Незера и Энда;
* контракты рыбака и кузнеца;
* донат-косметика и личные переключатели;
* антигриф и внутриигровые тикеты `/911`.
