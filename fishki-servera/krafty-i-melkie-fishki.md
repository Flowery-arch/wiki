# Крафты и мелкие фишки

На этой странице собраны механики, которые сейчас подтверждены исходниками `FLW_PLUGIN`. Старые сезонные или отключенные эксперименты сюда не включены.

## Кастомные крафты

### Глубинный сланец вместо булыжника

В этих рецептах можно использовать `колотый глубинный сланец` вместо обычного булыжника.

#### Наблюдатель

```text
C C C
R R C
C Q O

C = колотый глубинный сланец
R = редстоун
Q = кварц
O = компаратор
```

#### Раздатчик

```text
C C C
C R C
C B C

C = колотый глубинный сланец
R = редстоун
B = лук
```

#### Выбрасыватель

```text
C C C
C R C
C - C

C = колотый глубинный сланец
R = редстоун
- = пустая ячейка
```

#### Рычаг

```text
- - -
- C -
- S -

C = колотый глубинный сланец
S = палка
- = пустая ячейка
```

#### Печь

```text
C C C
C - C
C C C

C = колотый глубинный сланец
- = пустая ячейка
```

### Камнерез для глубинного сланца

Через камнерез работают две конверсии:

| Вход | Результат |
| --- | --- |
| `deepslate` | `cobbled_deepslate` |
| `cobbled_deepslate` | `deepslate` |

### Шерсть в нитки

Любой блок шерсти можно переработать в `4 нити`.

| Вход | Результат |
| --- | --- |
| 1 любая шерсть | 4 нити |

### Мох и моховой ковер

| Рецепт | Результат |
| --- | --- |
| 9 `moss carpet` | 1 `moss block` |
| 9 `moss block` | 1 `moss carpet` |

### Быстрая переделка бетона

Если бросить сухой бетон в котел с водой, он превратится в обычный бетон того же цвета.

Важно:

* работают все 16 цветов сухого бетона;
* котел должен быть хотя бы частично заполнен водой;
* один брошенный стак расходует один уровень воды в котле.

## Рамки

### Блокировка поворота предмета

Зажмите `Shift` и нажмите ПКМ по рамке `пчелиными сотами`.

Что происходит:

* поворот предмета в рамке блокируется;
* предмет из рамки по-прежнему можно вынуть;
* если рамку сломать или вынуть предмет, блокировка снимается.

### Превращение обычной рамки в светящуюся

Зажмите `Shift` и нажмите ПКМ по обычной рамке `светящимся чернильным мешком`.

Что сохраняется:

* предмет в рамке;
* направление рамки;
* видимость рамки;
* текущий поворот;
* блокировка поворота, если она уже была.

### Сделать рамку невидимой

Зажмите `Shift` и нажмите ПКМ по рамке `ножницами`.

Нюансы:

* в рамке уже должен лежать предмет;
* действие переключает видимость туда-обратно;
* ножницы получают износ вне креатива.

## Теги карт

Теги задаются прямо в названии карты при переименовании на наковальне.

### `#nocopy`

Запрещает копирование карты:

* на верстаке;
* на столе картографа.

На карте появляется пометка о запрете копирования.

### `#invframe`

Когда такая карта вставляется в рамку, сама рамка становится невидимой.

На карте появляется пометка о невидимой рамке.

### `#author`

Добавляет в описание карты строку с автором.

Формат:

```text
Автор: <ник игрока>
```

### Комбинирование тегов

Теги можно совмещать в одной карте, например:

```text
Моя карта #author #nocopy #invframe
```
