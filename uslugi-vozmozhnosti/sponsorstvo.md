# Спонсорство

На FLW.ONL поддержка проекта не должна ломать баланс. По текущим плагинам подтверждены в первую очередь косметические и сервисные механики, а не pay-to-win преимущества.

## FLW+

`FLW+` отображается в профиле игрока и используется как статус поддержки проекта.

Что подтверждено по текущим исходникам:

* статус FLW+ показывается в `/profile`;
* у сервера есть отдельный GUI доната и косметики;
* часть личных удобств и визуальных фишек управляется отдельными правами и переключателями;
* косметика не меняет баланс выживания.

## Что можно открыть в игре

| Команда | Что делает |
| --- | --- |
| `/donate` | Главное меню поддержки и косметики. |
| `/donate menu` | Открывает GUI с доступной косметикой. |
| `/donate wallet` | Даёт ссылку на страницу поддержки. |
| `/donate list <игрок>` | Показывает косметику другого игрока, если он в сети. |
| `/profile` | Показывает профиль игрока, включая статус FLW+. |

## Подтвержденные сервисные механики сервера

Ниже перечислены механики, которые реально существуют в текущих плагинах сервера:

* подсветка друзей;
* отключение фантомов;
* спидометр в action bar;
* bossbar Хранителя;
* редактирование описания предметов через `/itemlore`;
* привязка компаса и тотема к другу;
* личные переключатели через `/toggle`.

Важно:

* конкретный набор прав может меняться между сезонами;
* старые упоминания вроде `/setcolor` или `mixer burn` больше не считаются подтвержденной частью текущего FLW+ в этой wiki.
