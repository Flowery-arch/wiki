# Команды и чат

Ниже собраны основные команды, которые подтверждены текущими плагинами `FLWChat`, `FLW_PLUGIN` и `flw_worldb`.

## Важные команды

| Команда | Что делает |
| --- | --- |
| `/help` | Открывает меню навигации по проекту. |
| `/info` | Показывает основную информацию по серверу. |
| `/profile [игрок]` | Открывает профиль игрока со статистикой, FLW+ и вкладками сервера. |
| `/donate` | Открывает меню доната и косметики. |
| `/vote` | Показывает ссылку и кнопку голосования. |
| `/top <vote\|reputation\|playtime\|donations>` | Показывает топы сервера. |
| `/911 <описание>` | Отправляет тикет администрации прямо из игры. |

## Авторизация

| Команда | Что делает |
| --- | --- |
| `/register <пароль> <повтор>` | Регистрация на сервере. |
| `/login <пароль>` | Вход на сервер. |
| `/changepass <старый> <новый>` | Смена пароля. |

## Чат и общение

| Команда | Что делает |
| --- | --- |
| `сообщение` | Локальный чат. |
| `!сообщение` | Глобальный чат. |
| `/me <текст>` | Сообщение от третьего лица. |
| `/do <текст>` | RP-команда `/do`. |
| `/try <текст>` | RP-команда с процентным шансом. |
| `/tell <игрок> <текст>` | Личное сообщение. |
| `/msg <игрок> <текст>` | Алиас личного сообщения. |
| `/reply <текст>` | Быстрый ответ последнему собеседнику. |
| `/mail <игрок> <текст>` | Письмо игроку, когда он не в сети. |
| `/ignore <игрок>` | Игнор игрока. |
| `/ignorelist` | Список игнорируемых игроков. |
| `/afk` | Переключение режима AFK. |
| `/anon <текст>` | Анонимное сообщение. |

## Настройки чата

| Команда | Что делает |
| --- | --- |
| `/chatsetting` | Открывает меню настроек чата. |
| `/chatcolor` | Меняет цвета личного оформления чата. |
| `/symbol` | Открывает подбор символов и эмодзи. |
| `/sprite` | Открывает подбор спрайтов. |
| `/translateto <язык> <текст>` | Переводит сообщение на выбранный язык. |
| `/stream <ссылки>` | Помечает игрока как стримера и анонсирует ссылку. |

## Информация и поиск игроков

| Команда | Что делает |
| --- | --- |
| `/online <игрок>` | Показывает, в сети ли игрок, когда заходил и сколько играл. |
| `/ping [игрок]` | Показывает задержку. |
| `/home` | Показывает координаты кровати или дома и расстояние до них. |
| `/spawn` | Показывает координаты спавна и расстояние до него. |
| `/toponline` | Топ игроков по времени в игре. |

Важно:

* `/home` и `/spawn` показывают информацию.

## Друзья, репутация и Discord

| Команда | Что делает |
| --- | --- |
| `/rep up <ник>` | Повысить репутацию игрока. |
| `/rep down <ник>` | Понизить репутацию игрока. |
| `/link` | Показать статус привязки Discord. |
| `/link start` | Получить код для привязки Discord. |
| `/link unlink` | Отвязать Discord. |

Важно:

* для `/rep` нужна привязка Discord;
* голос за репутацию нельзя отдать самому себе;
* у репутации есть суточный кулдаун на одного игрока.

## Серверные механики

| Команда | Что делает |
| --- | --- |
| `/toggle` | Список персональных переключателей. |
| `/toggle <название>` | Переключает конкретную настройку. |
| `/city` | Открывает меню городов. |
| `/city requests` | Открывает заявки в города. |
| `/fine` | Показывает активные штрафы игрока. |
| `/fine pay <id> <сумма>` | Оплата штрафа. |

## Полезные подкоманды

### `/toggle`

Подтвержденные переключатели:

* `warden-boss-bar`
* `speedometer`
* `phantoms`
* `hide-other-particles`
* `hide-reputation-messages`

### `/donate`

| Команда | Что делает |
| --- | --- |
| `/donate` | Главное меню доната. |
| `/donate menu` | Открывает GUI с косметикой. |
| `/donate wallet` | Даёт ссылку на страницу поддержки. |
| `/donate list <игрок>` | Показывает косметику другого игрока, если он в сети. |

### `/fisher` и `/smith`

У NPC есть свои ветки подкоманд:

* просмотр контрактов;
* прогресс по активным заказам;
* сдача ресурсов;
* диалоговые окна;
* возврат к списку заказов.
