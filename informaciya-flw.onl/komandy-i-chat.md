# Команды и чат

### ▌ Начальные и важные команды <a href="#nachalnye-i-vazhnye-komandy" id="nachalnye-i-vazhnye-komandy"></a>



|                                            |                                    |
| ------------------------------------------ | ---------------------------------- |
| /register (Пароль) (Повтор Пароля)         | Зарегистрироваться на сервере.     |
| /login (Пароль)                            | Войти на сервер.                   |
| /changepass (Старый пароль) (Новый пароль) | Сменить пароль.                    |
| /help                                      | Открыть меню навигации по проекту. |

### ▌Скины <a href="#skiny" id="skiny"></a>

|                                  |                                                            |
| -------------------------------- | ---------------------------------------------------------- |
| /skin set (Никнейм Игрока)       | Поставить скин по нику какого-либо лицензионного аккаунта. |
| /skin url "(Ссылка на Картинку)" | Поставить скин по ссылке на картинку.                      |
| /skin clear                      | Очистить свой скин.                                        |
| /skin random                     | Поставить себе случайный скин.                             |

### ▌Чат и взаимодействие <a href="#chat-i-vzaimodeistvie" id="chat-i-vzaimodeistvie"></a>

|                                      |                                                                         |
| ------------------------------------ | ----------------------------------------------------------------------- |
| _сообщение_                          | Написать сообщение в локальный чат.                                     |
| !_сообщение_                         | Написать сообщение в глобальный чат.                                    |
| /me _сообщение от третьего лица_     | Написать сообщение от третьего лица (например: `*nickname сделал дом`). |
| /msg /m (Никнейм Игрока) _сообщение_ | Написать игроку в личные сообщения.                                     |
| /r _сообщение_                       | Быстро ответить последнему игроку.                                      |
| /mail (Никнейм Игрока) _сообщение_   | Написать игроку письмо _(может быть отправлено оффлайн-игроку)._        |
